# Realtime Chat App (Golang)

## Stack

- Go (Golang)
- WebSocket
- PostgreSQL

## Struktur Utama

- `cmd/server`: entrypoint backend Go.
- `internal`: API handler, service, dan model backend.
- `pkg`: utilitas umum.
- `docs`: frontend statis untuk GitHub Pages.
- `uploads`: penyimpanan file upload saat runtime backend.
